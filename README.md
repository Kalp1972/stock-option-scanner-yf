# Stock to Option Scanner (yfinance)

**Free, no-login, real-time stock scanner + option trade picker** using `yfinance`.

---

## Features
- Scan 10 NSE stocks with **Supertrend + RSI + Volume + ATR**
- Auto-pick **best tradable stock**
- Pull **live option chain**
- Rank & pick **best Long Call / Put / Cash-Secured Put**

---

## Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
